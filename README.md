# 天気アプリ（Weather App）

# アプリ紹介
都市名を入力し、現在の天気情報を取得 * 表示するAndroidアプリです。（OpenWeather API使用）

# 使用技術
- Java
- Android Studio
- OpenWeather API
- Glide

# 主な機能
- 都市検索（日本語対応）
- 現在の天気表示
- 天気アイコン表示
